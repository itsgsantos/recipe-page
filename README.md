## 🚀 Technical Features

* **Semantic HTML:** Proper use of `article`, `section`, `h1-h3`, and list tags for accessibility and SEO.
* **Mobile-First Workflow:** Designed for mobile devices first, then scaled up for desktop screens.
* **Modern CSS:** Utilization of CSS Custom Properties (variables) for consistent color and font management.
* **Typography Focus:** Careful implementation of Google Fonts to match the professional design provided.
* **Responsive Layout:** Fluid layout using Flexbox/Grid to ensure the component looks great on all screen sizes.

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** CSS3 (Flexbox, Custom Properties)
- **Design System:** Provided by Frontend Mentor

## 🧠 Key Learnings

Building this recipe page allowed me to refine my fundamental front-end skills. 

**Main Takeaways:**
- Working with **nested lists** (ordered and unordered) and styling markers.
- Implementing a professional **table structure** for nutritional data.
- Managing image responsiveness within a container to prevent layout shifts.
- Improving the use of **CSS Reset** to ensure cross-browser consistency.
